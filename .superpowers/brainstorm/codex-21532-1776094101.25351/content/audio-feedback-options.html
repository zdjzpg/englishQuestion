录音题反馈草图

先重点看“开始录音 / 录音中”的反馈，再顺带看“听示范 / 正在听”的感觉。你已经倾向“陪伴感 + 明显关键动效”，所以我把混合方案放在中间并标记为推荐。

竞品里常见的有效做法通常有 3 类：纯波形、纯吉祥物、吉祥物 + 关键动效叠层。对低龄孩子来说，第三种最容易理解“现在在听 / 现在该说了”。

方案 A：强游戏感

更像闯关
录音中GO!
🎙
Louder, louder!
开始录音：按钮放大 + 倒计时 3,2,1
录音中：强烈波形 + 分数环上涨
完成：啪一下弹出得分卡
优点：刺激、明确。缺点：容易太兴奋，像游戏，不够像小老师陪伴。

方案 B：混合推荐

推荐
录音中Listening Buddy
Great! Say it clearly.
听示范：小伙伴侧耳 + 音符飘起
开始录音：话筒亮起 + 小伙伴看向孩子
录音中：耳朵竖起 + 温和波形 + 呼吸光圈
优点：孩子能看懂“轮到我说了”，同时不会太吵。最适合你现在这种儿童测评气质。

方案 C：温柔陪伴感

最安静
正在听Soft
I am listening...
听示范：轻微摇摆 + 小音符
开始录音：按钮只换色，不强提醒
录音中：几乎没有波形，强调表情陪伴
优点：柔和。缺点：孩子未必立刻理解“现在可以开口说了”。
我建议的状态机

听示范 / 正在听

角色侧耳、头略微偏向音源、背景出现音符或声波纹。孩子一眼知道“先听”。

开始录音 / 录音中

角色转向孩子、话筒按钮发光、出现温和波形与一圈呼吸光，孩子一眼知道“轮到我说了”。
