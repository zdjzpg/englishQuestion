# 儿童英语测评项目上下文

注意：进入这个项目继续开发前，先读 [agent.md](D:\aa-workplace\EnglishQuestion\agent.md)。

最后更新时间：2026-04-17（腾讯云口语评测方案 B 已确认）

## 项目定位

这是一个面向儿童英语测评的双端项目：

- 老师端：登录后管理卷子、员工、答题记录
- 学生端：通过分享码进入卷子，填写信息后开始答题

当前整体目标已经从“能做题”升级到“更像儿童互动测评”，所以学生端页面在持续弱化后台感、试卷感，强化游戏感、贴纸感、奖励反馈和动画体验。

## 技术栈

- 前端：Vue 3 + Vue Router + Ant Design Vue
- 后端：Node.js + Express
- 数据库：MySQL + mysql2
- 动效：
  - `gsap`
  - `canvas-confetti`
- 导出：
  - `html-to-image`

## 核心产品规则

### 老师端

- 新建卷子默认是空白卷
- 卷子总分必须等于 100 分才允许保存
- `听音做指令` 题在保存前必须完成关键配置，否则不允许保存：
  - 已上传场景图
  - 已设置目标区域
  - 已设置正确答案
- 题目能力维度当前只使用：`听 / 说 / 读`
- 每题最多允许配置 2 个能力维度
- 已有答题记录的卷子不能直接编辑，只能复制后再改
- 报告评语的分数段不能重复
- 开启转盘抽奖时：
  - 每个奖品概率必须大于 0
  - 概率总和必须等于 100

### 学生端

- 不暴露老师端配置入口
- 学生信息必填：
  - 姓名
  - 手机号
  - 年龄
  - 年级 / 班级
- 学校字段已支持，但当前不是必填
- 默认不提前显示错误提示，点击“开始”后再提示未填项
- 题目页持续在做单屏适配，尽量让内容在 100vh 内完成展示
- 麦克风作答题当前交互规则：
  - 录音中可以手动结束录音
  - 题目页不显示识别文本
  - 题目页不显示“演示结果 / 演示评分 / 演示回答”按钮
  - 题目页不显示即时分数
  - 分数只在最终报告中体现

### 报告与奖励

- 每张卷子支持配置报告评语：
  - 开头评语
  - 中间评语（按分数命中）
  - 结尾评语
- 报告支持导出 HTML 和 PNG
- 报告页当前统一走平均两列布局：
  - 顶部 KPI 区：平均两列
  - 中部表现 / 雷达区：平均两列
  - 底部评语 / 今日收获区：平均两列
- 每张卷子可单独配置：
  - 开屏动画
  - 完成动画
  - 转盘抽奖
  - 奖品列表与概率
- 转盘抽奖当前规则：
  - 抽中的礼物使用独立结果弹层展示，不再把机器弹窗撑成长页
  - 奖品标签位置已改成围绕自身中心点定位，避免整体偏到扇区外侧
